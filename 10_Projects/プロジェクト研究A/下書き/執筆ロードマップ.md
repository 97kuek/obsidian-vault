# 執筆ロードマップ

研究ノート → 最終成果物への変換フロー。

## フロー

```
実験ノート ──┐
             ├→ 実験結果まとめ ──┐
論文メモ ───┘                    ├→ 執筆ドラフト → 最終成果物
             ┌→ 永続ノート ─────┘
勉強会ノート┘
```

1. **収集** — 実験ノート・論文メモを書く（素材の蓄積）
2. **統合** — [[実験結果まとめ]] と [[【MOC】永続ノート]] で素材を整理
3. **構成** — 執筆ドラフトでアウトラインを作る（`99_Templates/執筆ドラフト用.md`）
4. **執筆** — 本文を書く
5. **提出** — 最終成果物として完成

| 成果物 | 種別 | 期限 | 状態 | ファイル |
|---|---|---|---|---|
| 中間報告スライド | slide | - | 未着手 | - |
| 最終レポート | report | - | 未着手 | - |
