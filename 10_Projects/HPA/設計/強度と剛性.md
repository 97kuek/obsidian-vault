# 強度と剛性（HPA）

> 「壊れるか（強度）」と「どれだけ変形するか（剛性）」は別物。混同しやすいが、設計で効く場面も対策も異なる。
> 関連: [[飛行安定性]] / [[スパン荷重分布と誘導抗力]] / [[過年度空力設計]]

---

## 直感的なイメージ

| | 強度 | 剛性 |
|---|---|---|
| 問い | 壊れるか？ | どれだけ変形するか？ |
| 関心 | 破断・降伏の限界 | 変形・たわみの大きさ |

## 強度

- **「どれだけの力まで耐えられるか」**を表す性質。力を加えていくと、ある限界を超えたときに**破断・降伏**が起きる。その限界値が強度。
- 代表的な指標：
	- **引張強さ**：破断するときの最大応力
	- **降伏強さ**：永久変形が始まる応力
	- **疲労強さ**：繰り返し荷重に対する限界

## 剛性

- **「同じ力を加えたとき、どれだけ変形するか」**を表す性質。力と変形量の比で定義される。

$$k = \frac{F}{\delta}$$

- $k$：剛性（N/mm など）／$F$：力／$\delta$：変形量
- 剛性が**高い** → 変形しにくい（硬い）／**低い** → 同じ力でも大きく変形する（柔らかい）
- 材料の観点では、**ヤング率 $E$** が剛性の本質的な指標。

## 翼における強度と剛性

### 強度が問題になる場面

- **主桁の曲げ強度**：揚力による曲げモーメントに対して折れないか
- **翼リブの圧縮強度**：座屈しないか
- **かんざしの強度**：胴体-翼の結合部、桁の継ぎ手
- HPAではフライト中の突風や操作ミスで瞬間的に設計荷重を超えることがあり、これが強度の限界になりやすい。

### 剛性が問題になる場面

- **① 翼のたわみと空力弾性**：

```
荷重 → 翼がたわむ → 翼の迎角が変化 → 揚力分布が変わる → さらにたわむ
```

これが**ダイバージェンスやフラッター**につながる。剛性が低いと空力と構造が連成して不安定になる（→ [[飛行安定性]] の空力弾性）。

- **② 翼のねじれ剛性（トーショナルスティフネス）**：翼がねじれると**エルロン効果の逆転**が起きる。
	- 本来：エルロンを下げる → その翼の揚力増加 → 上昇
	- 剛性不足：エルロンを下げる → 翼全体がねじれて迎角減少 → むしろ下降
- **③ プロペラのたわみ**：

```
回転中のプロペラが曲がる → ピッチ角が変化 → 推力・効率が落ちる → さらに振動が発生
```

## 実際の設計フロー（概念）

```
1. 空力設計で揚力分布を決める
        ↓
2. 揚力分布から桁にかかる曲げモーメント・せん断力を計算
        ↓
3. 強度チェック：σ_max < σ_allowable（安全率込み）
        ↓
4. 剛性チェック：たわみ量 < 許容値、フラッター速度 > 最大飛行速度
        ↓
5. どちらかが満たせない → 断面形状・積層数を変更
        ↓
6. 重量チェック → 最初に戻る
```
