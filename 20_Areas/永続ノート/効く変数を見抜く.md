# 効く変数を見抜く

<!-- 下書き。骨子だけ置いた。自分の理解・言葉で書き直すこと。 -->

複雑に見える問題も、本当に効く独立な自由度はごく少数。**拘束で従属変数を消し、残った変数の中でも感度の大きいものに集中する**、という問題の絞り込み方。

## 核心的な主張

- 見かけのパラメータが多くても、それらは互いに独立ではない。多くは「ある拘束が active になった瞬間に値が決まる**従属変数**」。
- だから最初にやるべきは「変数を数える」ことではなく、**どの拘束が効いているかを見抜いて自由度を潰す**こと。
- 残った主自由度の中でも、**感度（結果への効き目）が大きい変数に労力を集中する**。効き目は多くの場合スケーリング則の**指数**に現れる。

## 詳細・補足（HPAの例）

- 人力機の設計変数は無数に見えるが、レイノルズ数・$Cl_{max}$・パイロット出力などの拘束に挟まれ、主自由度は実質スパン $b$・重量 $W$・翼面荷重 $W/S$ の3つに集約される。
- 最小所要動力 $P_\text{min} \propto W^{3/2} b^{-3/2} f^{1/4} e^{-3/4}$ を見ると、$W$ と $b$ は $3/2$ 乗で効くのに有害抗力 $f$ は $1/4$ 乗でしか効かない。
- → 「フェアリングで $f$ を削る（$1/4$乗）」より「重量を削りスパンを伸ばす（$3/2$乗）」が桁で効く、という**優先順位が指数から読める**。
- 出典: [[設計変数と最適化]]

## 他分野への広がり

- 最適化：効いている制約（active constraint）だけが解を支配する（→ [[双対性]] の影の価格）。
- 次元解析・スケーリング：べき則の指数の大小で支配項を判断する（流体・物理一般）。
- 一般：パレートの法則（少数の要因が大勢を決める）、ボトルネック思考。
- 共通する教訓：**全変数を均等に扱わない。拘束で減らし、感度で順位づけて、効く少数に集中する**。
