# TD学習（時間的差分学習）

「次の状態の見積もりを使って、今の状態の見積もりを少しずつ直す」という強化学習の中心的な考え方。

## 核心的な主張

状態 $s$ の価値 $V(s)$ は「$s$ 以降の報酬の総和の期待値」。だから $s$ から $s'$ へ遷移して報酬 $r$ を得たなら、

$$V(s) \approx r + V(s')$$

が成り立つはず。そこで現在の $V(s)$ を、この新しい見積もりに**少しだけ**近づける。

$$V(s) \leftarrow V(s) + \alpha\big( r + \gamma V(s') - V(s) \big)$$

括弧の中 $r + \gamma V(s') - V(s)$ が **TD誤差**。「予測」と「より新しい見積もり」のズレであり、これをゼロに近づけるのが TD 学習の本質。

## 詳細・補足

「＝」でなく「近づける」のは、$r + \gamma V(s')$ 自体もまだ正確でない（$V(s')$ も学習途中）から。少しずつ修正を繰り返して真の値へ収束させる。$\alpha$ は学習率、$\gamma$ は割引率（将来の報酬を割り引き、無限系列でも和を収束させる）。

TD 学習の強みは**状態遷移行列を一切使わない**こと。実際に動いて観測した状態を記録していけばよいので、未知環境にもほぼそのまま使える。
