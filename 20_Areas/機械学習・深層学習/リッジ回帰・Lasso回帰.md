## 線形回帰の復習と問題点

通常の線形回帰（最小二乗法）は、予測値と実測値の二乗誤差を最小化する。
$$\min_{w} \sum_{i=1}^{N} (y_i - w^\top x_i)^2 = \min_{w} |y - Xw|^2$$
これには主に二つの問題が生じる。
- **過学習：** 特徴量の数 $d$ がサンプル数 $N$ に対して大きい場合、モデルは訓練データのノイズまで拾ってしまい、新しいデータに対する予測性能が落ちてしまう
- **多重共線性：** 特徴量同士が強く相関していると、$X^\top X$ が特異（あるいはほぼ特異）になり、逆行列が不安定になる。結果として、重みの推定値が非常に大きくなったり符号が不自然に振れたりする。

正則化はこれらの問題に対処するために、**重みを不必要に大きくしないというペナルティを損失関数に加える**。
## リッジ回帰（L2正則化）

### 損失関数
リッジ回帰は、二乗誤差に**重みの二乗和**（L2ノルムの二乗）をペナルティとして加える。
$$L_{\text{Ridge}} = \sum_{i=1}^{N}(y_i - w^\top x_i)^2 + \lambda \sum_{j=1}^{d} w_j^2 = |y - Xw|^2 + \lambda |w|_2^2$$
- $\lambda > 0$ は正則化の強さを制御するハイパーパラメータ
- $\lambda$ が大きいほど重みは強く 0 に引き寄せられ、$\lambda = 0$ なら通常の線形回帰に戻る
### 直感的な意味
損失関数は「予測を正確にしたい」と「重みを小さくしたい」の二つの目的のバランスをとっている。モデルが訓練データに完璧に合わせようとして重みを大きくすると、ペナルティ項が増えて全体の損失が悪化する。結果として、データへの適合を少し犠牲にしてでも重みを穏やかに保つ解が選ばれる。これがバイアスとバリアンスのトレードオフで、バイアスをわずかに増やす代わりにバリアンス（推定の不安定さ）を大きく減らす。
### 閉形式の解
リッジ回帰には解析的な解がある。損失関数を $w$ で微分して 0 とおくと、
$$\frac{\partial L}{\partial w} = -2X^\top(y - Xw) + 2\lambda w = 0$$
整理すると、
$$w_{\text{Ridge}} = (X^\top X + \lambda I)^{-1} X^\top y$$
通常の最小二乗解 $(X^\top X)^{-1} X^\top y$ と比べると、$X^\top X$ の対角に $\lambda$ が足されている。これにより、$X^\top X$ がたとえ特異でも $X^\top X + \lambda I$ は必ず正則（逆行列が存在）になる。
### 重みに対する効果
リッジ回帰は全ての重みを一様に小さく縮小しますが、**ちょうど 0 にはしない。100個の特徴量があれば、100個すべてが小さな値を持った状態で残る。つまり特徴量の選択は行わない。
## Lasso回帰（L1正則化）
### 損失関数
Lassoは、ペナルティとして**重みの絶対値の和**（L1ノルム）を使う。
$$L_{\text{Lasso}} = \sum_{i=1}^{N}(y_i - w^\top x_i)^2 + \lambda \sum_{j=1}^{d} |w_j| = |y - Xw|^2 + \lambda |w|_1$$
### リッジとの本質的な違い：スパース性
Lassoの最も重要な性質は、**一部の重みをちょうど 0 にする**ことにある。これにより、不要な特徴量がモデルから自動的に排除される。つまりLassoは回帰と特徴量選択を同時に行う。
### なぜ L1 だと 0 になるのか
この違いは幾何学的に理解できる。
制約付き最適化の観点で考えると、リッジ回帰は「$|w|_2^2 \leq s$（円の内側）」という制約のもとで二乗誤差を最小化する問題と等価。Lassoは「$|w|_1 \leq s$（ひし形の内側）」という制約になる。
2次元の場合、二乗誤差の等高線は楕円状。最適解は等高線と制約領域が接する点。リッジの制約領域は円なので、接点は一般に座標軸上にはなく、どの重みも 0 にならない。一方、Lassoの制約領域はひし形で角が座標軸上にあるため、楕円の等高線はひし形の角に接しやすくなる。角に接するということは、少なくとも一つの重みが 0 であることを意味する。次元が高くなるほど、ひし形（超多面体）の角や辺が増えるため、多くの重みが 0 になりやすくなる。
### 最適化の方法
- L1ノルムの絶対値関数は $w_j = 0$ で微分不可能なので、リッジのように閉形式の解を求めることができまない。
- 代わりに、座標降下法（coordinate descent）が広く使われる。これは1つの重み $w_j$ だけを更新し、他を固定するという操作を全ての $j$ について巡回的に繰り返す方法です。各ステップでは**ソフト閾値処理**（soft thresholding）と呼ばれる操作が適用され、

$$w_j \leftarrow S\left(\frac{1}{N}\sum_{i=1}^{N} x_{ij} r_i,; \frac{\lambda}{2N}\right)$$

ここで $r_i$ は $w_j$ を除いた残差、$S$ はソフト閾値関数です。

$$S(z, \gamma) = \begin{cases} z - \gamma & (z > \gamma) \ 0 & (|z| \leq \gamma) \ z + \gamma & (z < -\gamma) \end{cases}$$

この関数の効果で、推定値の絶対値が閾値 $\gamma$ 以下なら重みがちょうど 0 に設定されます。
## 正則化パラメータ $\lambda$ の選び方
- 一般的には**交差検証**（cross-validation）を用いる
- データを $k$ 個のフォールドに分割し、$k-1$ 個で学習、残り1個で検証というのを $k$ 回繰り返して、検証誤差の平均が最小になる $\lambda$ を選ぶ
- $\lambda$ が小さすぎれば正則化が効かず過学習し、大きすぎればモデルが単純になりすぎて未学習になる
## 使い分けの指針
- リッジ回帰は、多くの特徴量がそれぞれ少しずつ予測に寄与しているケースに向いている。どの特徴量も捨てずに全体的に縮小する。多重共線性の解消にも適している。
- Lassoは、本当に重要な特徴量が一部だけで、残りは無関係だと考えられるケースに向いている。不要な特徴量を自動で除去し、解釈しやすいモデルを得られる。
## まとめ
リッジ回帰とLasso回帰はどちらも**重みにペナルティを課す**という同じ発想に基づいているが、ペナルティの形（L2 か L1 か）の違いが、「全体を縮小する」か「一部を 0 にする」かという質的に異なる振る舞いを生みだす。この違いは制約領域の形状（円 vs ひし形）から幾何学的に理解でき、正則化の理論において最も重要な対比の一つ。
