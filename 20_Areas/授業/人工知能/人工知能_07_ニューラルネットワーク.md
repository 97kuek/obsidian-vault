# 人工知能：ニューラルネットワークと深層学習

- 親ノート: [[人工知能]]
- 関連ノート: [[離散数学_04_ニューラルネットワークと計算グラフ]]


## ニューラルネットワークの基本概念

### 生物の神経系からの発想

- ニューラルネットは、**生物の神経系が相互に結ばれた複雑なネットワークを構成している**ことをヒントに考案された学習法である
- 生物のニューロンの動作は次のようにモデル化される
    - ニューロンは複数の入力繊維と出力繊維をもつ
    - 電気パルスで刺激が与えられ、シナプスで化学信号に変換される
    - その内容に応じて次のニューロンへ電気パルスとして伝わる
- この「単純な計算を行うユニットが多数結合してネットワークを構成する」という構造が基本の考え方
- それぞれの結びつきには**重み**が定義され、結合の強さを表す

### ニューロンの数理モデル

- 各ユニットの動作は、次の2段階からなる
  - **荷重和の計算**：他のユニットからの入力 $x_i$ に重み $w_i$ を掛けて総和をとり、閾値 $\theta$ を引く。 $$y = \sum_{i = 1}^{n} w_i x_i - \theta$$
  - **活性化関数の適用**：荷重和 $y$ に関数 $f$ を施し、出力 $x = f(y)$ とする。これを次のユニットへ渡す。

- ここで $w_i$ をシナプスの結合荷重、$\theta$ を閾値、$f$ を活性化関数と呼ぶ

### 論理ゲートの実現とXOR問題

- 1つのニューロンとステップ関数を使うと、基本的な論理演算を実現できる
- 重みと閾値を調整すればよい
    - **AND**：$w_1 = w_2 = 1, \ \theta = 1.5$（両方1のときだけ出力1）
    - **OR**：$w_1 = w_2 = 1, \ \theta = 0.5$（どちらか1なら出力1）
    - **NOT**：$w_1 = -1, \ \theta = -0.5$（入力を反転）
- 1つのニューロンでは **XORは表現できない**。
    - XORは「直線1本で分離できない（線形分離不可能）」問題であり、単一ニューロンの表現力を超える
    - この限界こそが、後の**多層化**の動機となる

### 活性化関数の種類

|関数|定義|特徴|
|---|---|---|
|ステップ関数 $s(x)$|$x>0$ で1、$x\le 0$ で0|生物の「全か無かの法則」に近く直感的。ただし**不連続で微分不可能**|
|シグモイド関数|$\sigma(y) = \dfrac{1}{1+\exp(-y)}$|**微分可能**で計算が容易。誤差逆伝播法で用いる|
|tanh（原点通過型）|$\tanh(y) = 2\sigma(2y)-1$|原点を通る形が望ましいという考えから、シグモイドを線形変換したもの|
|ReLU|$R(y) = \max(0, y)$|0付近で連続微分ではないが、$y>0$ で微分が定数1。**勾配喪失を防ぐ**。ランプ関数とも呼ぶ|
|ソフトプラス|$\log(1+e^{y})$|ReLUを近似し、連続微分可能にしたもの|

- 初期にはステップ関数が普通に使われていたが、微分できないため学習アルゴリズムに不向き
- 微分可能なシグモイド関数の導入が、後述の誤差逆伝播法を可能にした
- 深層化が進むと、シグモイドの欠点（勾配喪失）を避けるためにReLUが主流となった

### ネットワークの種類

- **階層型（フィードフォワード型）ニューラルネット**
    - 入力層 → 中間層 → 出力層と、信号が**一方向**に流れる。

- **再帰型（リカレント型／RNN）**
    - **ループ（フィードバック）**をもつ。前の時刻の出力の一部を次の入力の一部として使う。
    - 文章のように**順序（時系列）がある**データに使われる

### ネットワークの学習とは

- 入力に対して望む出力が得られるように、シナプスの結合荷重 $w_{ij}$ を変更していくことが「学習」
- この講義の数値例では、減衰係数 $\alpha$ と学習率 $k$ を使って次のように更新する
- $$w_{ij}(t+1)=\alpha w_{ij}(t)+\Delta w_{ij}(t),\qquad
- \Delta w_{ij}(t)=k\,\delta_j x_i$$
- $\alpha$ は既存の重みをどれだけ残すか、$k$ は更新量、$x_i$ は結合へ入る入力を表す
- $\delta_j$ は**学習信号**と呼ばれる
- この $\delta_j$ の中身が、学習法の違い（誤り訂正学習か、誤差逆伝播法か）を決める。

### 教師なし学習

#### Hebb学習

- 基本的なHebb則は、入力側の活動 $x_i$ と出力側の活動 $y_j$ が同時に大きい結合を強める

$$
\Delta w_{ij} = \eta x_i y_j
$$

- 「一緒に発火するニューロン同士の結合が強まる」という相関学習である
- **類似した特定パターンの出力が学習され、反応しやすくなる**
- 結果としてクラスタリングのような分類・抽出ができる
    - これはTopic 6で学んだクラスタリング（K-meansなど）と目的が近い
    - 教師信号なしに似たものをまとめる発想
- そのままだと荷重が大きくなりすぎるため、工夫が必要

### 自己組織化写像

- **高次元データを、たとえば2次元平面上に、分類ごとにまとめて配置する**手法。
    - 例：動物の特徴 → 肉食獣・草食獣に分かれる
    - 健康診断の数値（身長・体重・血圧・血糖値…）→ 「やせ形で尿酸値が高い」といった同傾向の人をグループ化
    - 言語のもつ類似性の可視化など
- 長所：**視覚的に捉えられる**
- 短所：通常は1次元または2次元の格子へ写像するため、高次元構造を完全には保存できず、初期値やハイパーパラメータにも依存する
- **アルゴリズムの骨子**（ニューラルネットを使う場合）
- 入力は $n$ 次元で $N$ 個あるとする。$m$ 個のユニットを通常は2次元格子上に並べ、各ユニットに入力と同じ $n$ 次元の重みベクトル $\boldsymbol{w}_i$ を持たせる。$m>n$ である必要はない
- 出力ノード間に距離 $\text{dist}(s_i, s_j)$ を定義する（グリッド上のユークリッド距離など）。
- 結合荷重 $w_{ij}(0)$ を $(0, 1)$ の範囲でランダムに初期化する。
- 入力 $\boldsymbol{x}_t$ を与え、重みベクトルが最も近いユニットを勝者 $k$ とする

$$
k = \arg\min_i \lVert\boldsymbol{x}_t-\boldsymbol{w}_i(t)\rVert
$$

- 勝者に近いユニットほど大きく更新する
- $$\boldsymbol{w}_i(t+1)=\boldsymbol{w}_i(t)
- +\eta(t)\,h_{k,i}(t)\bigl(\boldsymbol{x}_t-\boldsymbol{w}_i(t)\bigr)$$

    - $\eta(t)$ は学習率、$h_{k, i}(t)$ は格子上で勝者から離れるほど小さくなる近傍関数であり、通常は学習とともに小さくする
    - 勝者に近いノードほど強く、勝者に似た向きへ引き寄せられる。これにより「近いノードは似た特徴を担う」配置が自己組織的に形成される。

### 教師あり学習

#### 誤り訂正学習（パーセプトロン）

- F. Rosenblattが1957年に提案
- 各入力と出力の組を**教師データ**として与え、その信号に合うようニューロンを学習させる
- 標準的な**単純パーセプトロン**は、入力を線形結合して1つの出力を得る線形分類器である。多層ネットワークとは区別する
- 学習信号を **$\delta = d - x$**と定義する。
    - 実際の出力が教師データより小さければ $\delta>0$ となり、$x_i>0$ の重みは増える（出力を増やす方向へ修正）
    - $x_i<0$ なら逆

- **具体的な計算例**（2入力ニューロン）
- 初期値：$w_1(0) = 0.0, \ w_2(0) = 0.2, \ \theta = 0.5, \ \alpha = 0.9, \ k = 0.2$
- 教師データ：$x_1 = 0, \ x_2 = 1, \ d = 1$（これを何回か入力）
- ステップ関数 $s$ を使って計算すると：

|$t$|荷重和 $y = w_1x_1+w_2x_2-\theta$|出力 $x = s(y)$|更新後 $w_2$|
|---|---|---|---|
|0|$s(-0.3)$|0（教師と不一致）|$w_2(1) = 0.38$|
|1|$s(-0.12)$|0（不一致）|$w_2(2) = 0.542$|
|2|$s(0.042)$|**1（教師と一致）**|$w_2(3) = 0.4878$|

- $t = 2$ でいったん教師データと一致するが、その次のステップではまた一致しなくなり、**繰り返すうちに安定していく**
- この例で正解後にも $w_2$ が $0.542 \to 0.4878$ と変わるのは、更新式に減衰項 $\alpha w(t)$ が含まれるためである。標準的なパーセプトロン更新則では、正しく分類した場合の更新量は0になる
- 本来 $w_1 = 0$ のような重みは望ましくないが、この例では計算を簡単にするために用いている

#### なぜ多層が必要か

- パーセプトロンの誤り訂正学習は、**最後の層の結合荷重だけを学習する一層分の学習**であり、学習能力が低い
- 単層パーセプトロンは線形分離可能な問題しか表現できず、XORを扱えない。この限界が1969年にMinskyとPapertによって体系的に論じられた
- 解決策：**中間層を増やし、すべての層で学習する**
- これが**誤差逆伝播法**

#### 誤差逆伝播法

- 誤差逆伝播法の考え方には先行研究があり、Rumelhartらの1986年の研究によって多層ニューラルネットの学習法として広く普及した
- 活性化関数と損失関数が微分可能なら適用でき、シグモイドに限定されない
- 教師データは出力側で与えられるので、その値に合うように出力層のニューロンを学習し、それに基づいて1つ前の層を学習…と、**誤差を出力側から入力側へ逆向きに伝えていく**
- **手順の全体像**

1. 荷重を0に近い小さな乱数で初期化
2. 入力を入力層に与え、各層の出力を前から順に計算する
    - 中間層：$q_j = \sum_i w_{ij}x_i-\theta, \quad x_j = f(q_j)$
    - 出力層：$q_k = \sum_j v_{jk}x_j-\theta, \quad y_k = f(q_k)$
3. 出力と教師データの二乗和誤差 $E$ を計算 $$E = \frac{1}{2}\sum_{k = 1}^{L}(y_k-d_k)^2$$
4. $E$ を最小化するよう、勾配降下法で出力層の荷重 $v_{jk}$ を修正する

- $$\Delta v_{jk}=-\eta\frac{\partial E}{\partial v_{jk}}
- =\eta(d_k-y_k)f'(q_k)x_j$$
    - ここで $\delta_k = (d_k-y_k)f'(q_k)$ と定義
    - $f$ がシグモイドなら $f'(q_k) = \sigma(q_k)\big(1-\sigma(q_k)\big)$
5. 連鎖律を使い、中間層の誤差信号を求める

$$
\delta'_j = f'(q_j)\sum_k \delta_k v_{jk}
$$

- シグモイドの場合は

$$
\delta'_j = \sigma(q_j)\bigl(1-\sigma(q_j)\bigr)\sum_k\delta_kv_{jk}
$$
6. これを入力層に達するまで繰り返し、さらに各教師データについて繰り返す

- **特徴**：パーセプトロンと違い、すべてのニューロン間の荷重を学習するので学習能力が高い。
- **課題**：学習時の計算量は大きいが、一度学習すれば認識などに使える。問題の複雑さによっては**大量のデータ**が必要。音声・文字・画像認識で効果が認められている。

### 深層学習の概要

#### なぜ「ただ層を増やすだけ」ではうまくいかなかったか

- 誤差逆伝播法のネットワークは長らく3〜4層程度が主流で、それでも相応の認識率があった
- 2000年前後、計算機の能力向上で「さらに多層化すれば学習能力も上がる」と期待されたが、**必ずしもうまくいかなかった**
- 主な原因は次の通り
  - **勾配喪失**
      - 誤差逆伝播では、微分したシグモイド関数の積が層ごとに掛け合わされる。
      - シグモイドの微分は**最大でも0.25**で、$|y|$ が大きいとさらに0に近づく。これを何段も掛けると、勾配は数層で**ほぼ0に消えてしまう**
        - 逆に調整の仕方によっては勾配が爆発的に増大することもある（勾配爆発）
  - **過学習**
      - 教師データに合わせすぎ、少し違うデータでは正解が出なくなる現象。
      - 本来は多少一般化して未知データにも対応してほしいが、判定が過度に厳格になる。
  - **重みの初期値への強い依存**
      - 初期値によって勾配喪失の度合いが変わり、結果が大きく変動する
      - 良い初期値を試行錯誤で探すのは容易でない。

#### 対策1：ReLUの活用

- シグモイドの代わりにReLUを使う
- $y>0$ では微分が1なので、シグモイドより勾配を伝えやすく、勾配喪失を**軽減**できる
- $y\le0$ では勾配が0になるため、ユニットが更新されなくなるdead ReLUの問題もある

#### 対策2：積層オートエンコーダ（自己符号化器）

- 重みの初期値を**あらかじめ学習しておく**手法
- 特に入力層に近い部分で効果がある
    - 最初の2層を取り出し、その**鏡像構造**を加えて3層のネットワークを作る。
    - 入力と同じデータが出力に再現されるように、誤差逆伝播法で重みを学習する（＝情報が失われていないことを意味する）。
    - 得られた重みを保存し、これを後方の層へ繰り返していく。
- 意義：勾配喪失と初期値の問題を軽減できる。さらに、徐々にノード数を減らすことで**次元圧縮・特徴の集約表現**が得られる点も重要である。
    - これはTopic 6のPCA（次元圧縮）と発想が通じる。オートエンコーダは現在も表現学習や生成モデルで使われるが、深層学習全体がこの事前学習法を必須とするわけではない

#### 深層学習の特徴と現状の課題

- **特徴**
- 非線形の特徴抽出が可能
- 適切なデータ拡張や正則化によってノイズへの頑健性を高められるが、分布外データや敵対的摂動に常に強いわけではない
- 手作業の規則記述を減らせる一方、データ設計、損失関数、モデル構造、評価方法は人間が定める必要がある
- 学習の計算量は莫大だが一度学習すれば高い能力を使える
- 学習能力は分野（画像・音声・自然言語・対話など）に依存する
- **共通の課題**
- 過学習が頻繁に起こる、パラメータが多すぎて構成の工夫が必要
- 多大な計算資源と時間が要る
- **学習できてもその理由の解析が難しい**ため、いつ・なぜ間違うかの予測がつきにくい
- **フレームワーク**
- PyTorch、TensorFlow、JAXなど
- 大規模モデルではGPUなどの計算資源が有効だが、小規模モデルの学習・推論はCPUでも可能である
