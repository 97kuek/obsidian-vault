# Topic 4：決定木

## 阪神ファンの決定木

- **データ**

|id|性別|出身|学部|趣味|阪神ファン|
|---|---|---|---|---|---|
|A|男性|関東|工学部|スポーツ|×|
|B|女性|関東|工学部|スポーツ|×|
|C|男性|関西|工学部|スポーツ|○|
|D|男性|中部|理学部|スポーツ|○|
|E|男性|中部|法学部|旅行|○|
|F|女性|関西|法学部|旅行|○|
|G|男性|関東|法学部|旅行|○|
|H|男性|中部|理学部|旅行|○|
|I|女性|関東|理学部|旅行|○|
|J|男性|関西|工学部|旅行|○|
|K|女性|中部|理学部|スポーツ|×|

- ファン(○)が8人、非ファン(×)が3人なので、全体のエントロピーは 以下の通り
$$
E(S) = -\tfrac{8}{11}\log_2\tfrac{8}{11} - \tfrac{3}{11}\log_2\tfrac{3}{11} = 0.8453
$$

### 問題

> 性別・学部・趣味で分類したときの情報量を計算し、ルートに置くべき属性 $c$ を求めよ

### 解答

- 各属性で分割し、「分割後の重み付きエントロピー」を求めて $E(S) = 0.8453$ から引くと情報量が出る。情報量が大きいもので分類する
- **出身**（関東 {A,B,G,I}、中部 {D,E,H,K}、関西 {C,F,J}）
- $E(\text{関東})$：ファン2・非2 → $1.0$
- $E(\text{中部})$：ファン3・非1 → $0.8113$
- $E(\text{関西})$：ファン3・非0 → $0$
- 重み付き $= \tfrac{4}{11}(1.0)+\tfrac{4}{11}(0.8113)+\tfrac{3}{11}(0) = 0.6586$
- **情報量 $= 0.8453 - 0.6586 = 0.1867$**
- **性別**（男性 {A,C,D,E,G,H,J}、女性 {B,F,I,K}）
- $E(\text{男性})$：ファン6・非1 → $0.5917$
- $E(\text{女性})$：ファン2・非2 → $1.0$
- 重み付き $= \tfrac{7}{11}(0.5917)+\tfrac{4}{11}(1.0) = 0.7402$
- **情報量 $= 0.1051$**
- **学部**（工 {A,B,C,J}、理 {D,H,I,K}、法 {E,F,G}）
- $E(\text{工})$：ファン2・非2 → $1.0$
- $E(\text{理})$：ファン3・非1 → $0.8113$
- $E(\text{法})$：ファン3・非0 → $0$
- 重み付き $= 0.6586$（出身と同じ構造）
- **情報量 $= 0.1867$**
- **趣味**（スポーツ {A,B,C,D,K}、旅行 {E,F,G,H,I,J}）
- $E(\text{スポーツ})$：ファン2・非3 → $0.9710$
- $E(\text{旅行})$：ファン6・非0 → $0$
- 重み付き $= \tfrac{5}{11}(0.9710)+\tfrac{6}{11}(0) = 0.4413$
- **情報量 $= 0.8453 - 0.4413 = 0.4040$**

| 属性     | 情報量        |
| ------ | ---------- |
| 出身     | 0.1867     |
| 性別     | 0.1051     |
| 学部     | 0.1867     |
| **趣味** | **0.4040** |

- → **ルート属性は趣味**

### 問題

> 求めた属性で $S$ を分割し、各 $S_i$ を求めよ。

### 解答


- 趣味で分割すると、
- $S_{\text{旅行}} = {E, F, G, H, I, J}$ → **全員ファン(○)**。純粋なので葉ノードで確定。
- $S_{\text{スポーツ}} = {A, B, C, D, K}$ → ファン2(C,D)・非3(A,B,K)。**混在しているので再帰が必要**。


### 問題

> 各 $S_i$ について残りの属性で情報量を計算し、適切な属性 $c_i$ を求め、決定木を完成させよ。

### 解答

- 再帰の対象は $S_{\text{スポーツ}} = {A, B, C, D, K}$ のみ（残り属性は 性別・出身・学部）。 このときのエントロピーは $E(S_{\text{スポーツ}}) = 0.9710$。
- **性別**：男性{A,C,D}（ファン2・非1, $E = 0.9183$）、女性{B,K}（非2, $E = 0$） 重み付き $= \tfrac{3}{5}(0.9183) = 0.5510$ → 情報量 $0.4200$
- **出身**：関東{A,B}（$E = 0$）、関西{C}（$E = 0$）、中部{D,K}（ファン1・非1, $E = 1.0$） 重み付き $= \tfrac{2}{5}(1.0) = 0.4000$ → **情報量 $0.5710$** ← 最大
- **学部**：工{A,B,C}（$E = 0.9183$）、理{D,K}（$E = 1.0$） 重み付き $= 0.9510$ → 情報量 $0.0200$
- → ここでは **出身** で分割。
- 関東{A,B} → 非ファン(×) で確定
- 関西{C} → ファン(○) で確定
- 中部{D,K} → D(○)・K(×) で混在 → さらに再帰
- 最後に {D,K} を残り属性（性別・学部）で分ける。学部は両者とも理学部で分けられないため使えない。**性別**なら D(男性)→○、K(女性)→× と完全に分かれる。
- **完成した決定木**

```
趣味 ?
├─ 旅行   → ○ ファン            （E,F,G,H,I,J）
└─ スポーツ → 出身 ?
              ├─ 関東 → × 非ファン  （A,B）
              ├─ 関西 → ○ ファン    （C）
              └─ 中部 → 性別 ?
                        ├─ 男性 → ○ ファン   （D）
                        └─ 女性 → × 非ファン （K）
```

- 11人すべてを正しく分類できることを確認済みである。

## CART（ジニ係数）とエントロピーの比較

- **データ**
- 文系A・理系各100人、計200人に対して以下のようなデータが取れた

|指標|A(文系)|B(理系)|
|---|---|---|
|英語70点以上|60|30|
|英語50点以上|90|60|
|数学80点以上|15|75|
|数学60点以上|40|95|

- 分割前のジニ係数 $G(S) = 1 - (\tfrac{100}{200})^2 - (\tfrac{100}{200})^2 = 0.5$

### 問題

> その他の指標についてもジニ係数の減少量を求め、どの指標で分類するとよいかを考えよ。

### 解答

- 英語が50点以上で分類したとき
  - $G_{\text{以上}} = 1-(\tfrac{90}{150})^2-(\tfrac{60}{150})^2 = 0.48$
  - $G_{\text{未満}} = 1-(\tfrac{10}{50})^2-(\tfrac{40}{50})^2 = 0.32$
  - 重み付き $= \tfrac{150}{200}(0.48)+\tfrac{50}{200}(0.32) = 0.44$
  - **減少量 $= 0.5 - 0.44 = 0.0600$**

- 数学が80点以上で分類したとき
  - $G_{\text{以上}} = 1-(\tfrac{15}{90})^2-(\tfrac{75}{90})^2 = 0.2778$
  - $G_{\text{未満}} = 1-(\tfrac{85}{110})^2-(\tfrac{25}{110})^2 = 0.3512$
  - 重み付き $= \tfrac{90}{200}(0.2778)+\tfrac{110}{200}(0.3512) = 0.3182$
  - **減少量 $= 0.5 - 0.3182 = 0.1818$**

- 数学60点以上で分類したとき
  - $G_{\text{以上}} = 1-(\tfrac{40}{135})^2-(\tfrac{95}{135})^2 = 0.4170$
  - $G_{\text{未満}} = 1-(\tfrac{60}{65})^2-(\tfrac{5}{65})^2 = 0.1420$
  - 重み付き $= \tfrac{135}{200}(0.4170)+\tfrac{65}{200}(0.1420) = 0.3276$
  - **減少量 $= 0.5 - 0.3276 = 0.1724$**

- したがって、**数学80点以上** で分類するのが最良

### 問題

> 「英語70点以上(未満)」で分割したときの情報量を求めよ。

### 解答

- 分割前 $E(S) = -\tfrac{100}{200}\log_2\tfrac{100}{200}-\tfrac{100}{200}\log_2\tfrac{100}{200} = 1.0$
- 70点以上（A60/B30=90人）：$E = -\tfrac{60}{90}\log_2\tfrac{60}{90}-\tfrac{30}{90}\log_2\tfrac{30}{90} = 0.9183$
- 70点未満（A40/B70=110人）：$E = -\tfrac{40}{110}\log_2\tfrac{40}{110}-\tfrac{70}{110}\log_2\tfrac{70}{110} = 0.9457$
- 重み付き $= \tfrac{90}{200}(0.9183)+\tfrac{110}{200}(0.9457) = 0.9333$
- **情報量 $= 1.0 - 0.9333 = 0.0667$**

### 問題

> 同様にエントロピーを使い、最初にどの指標で分類するとよいかを求めよ。

### 解答


- **英語50点以上**：$E_{\text{以上}} = 0.9710$、$E_{\text{未満}} = 0.7219$ 重み付き $= \tfrac{150}{200}(0.9710)+\tfrac{50}{200}(0.7219) = 0.9087$ → 情報量 $0.0913$
- **数学80点以上**：$E_{\text{以上}} = 0.6500$、$E_{\text{未満}} = 0.7732$ 重み付き $= \tfrac{90}{200}(0.6500)+\tfrac{110}{200}(0.7732) = 0.7178$ → 情報量 $0.2822$
- **数学60点以上**：$E_{\text{以上}} = 0.8767$、$E_{\text{未満}} = 0.3912$ 重み付き $= \tfrac{135}{200}(0.8767)+\tfrac{65}{200}(0.3912) = 0.7189$ → 情報量 $0.2811$
- → エントロピー基準でも **数学80点以上** が最良
