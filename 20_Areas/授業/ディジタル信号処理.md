## 0. 全体マップ

```
離散時間信号 x[n]
│
├─ 理論的な周波数解析
│   └─ DTFT（離散時間フーリエ変換）
│       ↓ z = e^{jω} の特殊ケース
│
├─ システム解析・安定性
│   └─ z変換
│
├─ コンピュータで実計算
│   └─ DFT（離散フーリエ変換） ← FFTで高速化
│       ↑ DTFTをN点サンプリングしたもの
│
└─ 時間変化する信号の解析
    └─ STFT（短時間フーリエ変換） ← DFTを窓ごとに適用
        └─ 窓関数の選択が重要
```

## サンプリング定理・エイリアシング

### サンプリング定理

連続信号 $x(t)$ をサンプリング周波数 $f_s$ でサンプリングするとき：

$$x[n] = x(nT_s), \quad T_s = \frac{1}{f_s}$$

> [!important] ナイキスト条件 $$f_s > 2 f_{\max}$$ 信号の最大周波数 $f_{\max}$ の **2倍超** でサンプリングしないと元の信号を復元できない。 $f_N = f_s / 2$ を **ナイキスト周波数** という。

### エイリアシング（折り返し雑音）

$f_s < 2f_{\max}$ のとき、高周波成分が低周波成分に「折り返して」重なる現象。

| 状況                | 結果             |
| ----------------- | -------------- |
| $f_s > 2f_{\max}$ | 正常に復元可能        |
| $f_s = 2f_{\max}$ | 境界ケース（実用上は不十分） |
| $f_s < 2f_{\max}$ | エイリアシング発生 → 歪み |
### 対策：アンチエイリアシングフィルタ

サンプリング前にLPFで $f_{\max} < f_s/2$ になるよう帯域制限する。
## 2. DTFT（離散時間フーリエ変換）

### 定義

$$X(e^{j\omega}) = \sum_{n=-\infty}^{\infty} x[n] e^{-j\omega n}$$

$$x[n] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(e^{j\omega})e^{j\omega n} d\omega$$

### 変数 $\omega$ について

- $\omega$：**正規化角周波数**（単位：rad/sample）
- 範囲：$-\pi \leq \omega \leq \pi$（周期 $2\pi$）
- 実際の周波数との関係：$\omega = 2\pi f / f_s = 2\pi f T_s$

|$\omega$|対応する周波数|
|---|---|
|$0$|直流（DC）|
|$\pm\pi$|ナイキスト周波数 $f_s/2$|
### 特徴

|項目|内容|
|---|---|
|入力|離散・無限長の $x[n]$|
|出力|**連続スペクトル**（$\omega$ の関数）|
|周期性|$X(e^{j\omega})$ は $2\pi$ 周期|
|計算機で直接計算|❌ 不可（無限和・連続出力）|
### 主な使い方

- フィルタ設計の**理論的基礎**
- 周波数特性の定義（後述）
- DFT・z変換の特殊ケースとして位置づけ
## 3. z変換

### 定義

$$X(z) = \sum_{n=-\infty}^{\infty} x[n], z^{-n} \quad z \in \mathbb{C}$$

$$x[n] = \frac{1}{2\pi j} \oint X(z) z^{n-1} dz$$
### DTFTとの関係

$$z = e^{j\omega} \text{ のとき} \quad X(z)\big|_{z=e^{j\omega}} = X(e^{j\omega})$$

> [!note] 単位円 複素平面上の単位円（$|z|=1$）上でのz変換 ＝ DTFT
### 収束領域（ROC: Region of Convergence）

$X(z)$ が収束する $z$ の領域。安定性の議論に必須。

|ROCと安定性|条件|
|---|---|
|因果的・安定なシステム|すべての極が単位円**内部**にある|
|不安定|極が単位円**外部**|
|臨界安定|極が単位円**上**|
### 伝達関数 $H(z)$

LTIシステムの入出力関係：

$$Y(z) = H(z) \cdot X(z)$$

$$H(z) = \frac{Y(z)}{X(z)} = \frac{\sum_{k=0}^{M} b_k z^{-k}}{1 + \sum_{k=1}^{N} a_k z^{-k}}$$
### 主な使い方

- **システムの安定性判定**（極の位置を見る）
- **フィルタ設計**（極・零点の配置）
- 差分方程式の解析
- DTFTの一般化（成長信号にも適用可能）
## 4. 周波数特性

### 定義

伝達関数 $H(z)$ を単位円 $z = e^{j\omega}$ 上で評価したもの：

$$H(e^{j\omega}) = H(z)\big|_{z=e^{j\omega}}$$
これはDTFTそのもの。
### 分解

$$H(e^{j\omega}) = |H(e^{j\omega})| \cdot e^{j\angle H(e^{j\omega})}$$

|成分|名称|意味|
|---|---|---|
|$\|H(e^{j\omega})\|$|**振幅特性**（ゲイン特性）|各周波数をどれだけ通すか|
|$\angle H(e^{j\omega})$|**位相特性**|各周波数の位相ずれ|
### 群遅延

$$\tau(\omega) = -\frac{d}{d\omega} \angle H(e^{j\omega})$$

- 各周波数成分の**遅延時間**
- 群遅延が一定（**線形位相**）→ 波形の形が崩れない
- FIRフィルタは線形位相を実現しやすい
## 5. DFT（離散フーリエ変換）

### 定義

$N$ 点の信号 $x[n]$（$n=0,\ldots,N-1$）に対して：

$$X[k] = \sum_{n=0}^{N-1} x[n], W_N^{kn}, \quad k = 0, 1, \ldots, N-1$$

$$x[n] = \frac{1}{N} \sum_{k=0}^{N-1} X[k], W_N^{-kn}$$

ここで $W_N = e^{-j2\pi/N}$（回転因子、twiddle factor）
### DTFTとの関係

DFT はDTFTを $N$ 点等間隔サンプリングしたもの：

$$X[k] = X(e^{j\omega})\bigg|_{\omega = \frac{2\pi k}{N}}$$

|$k$|対応する正規化角周波数 $\omega$|対応する実周波数|
|---|---|---|
|$0$|$0$|$0$ Hz|
|$1$|$2\pi/N$|$f_s/N$|
|$N/2$|$\pi$|$f_s/2$（ナイキスト）|
|$N-1$|$2\pi(N-1)/N \approx 2\pi$|$\approx f_s$|

### 注意事項

> [!warning] DFTの暗黙の仮定 DFTは入力信号が **$N$ 点で周期的** であると仮定する。 実際には有限長の信号を切り出しているため、**周期的でない場合はスペクトル漏れ（leakage）が生じる**。
### FFT（高速フーリエ変換）

DFTを効率的に計算するアルゴリズム。計算量：

|方法|計算量|
|---|---|
|素朴なDFT|$O(N^2)$|
|FFT（Cooley-Tukey等）|$O(N \log N)$|

$N$ が2のべき乗のとき特に効率的。

### 主な使い方

- **実際の信号のスペクトル解析**（コンピュータで計算可能）
- 畳み込みの高速計算（線形畳み込み→FFT→逆FFT）
- フィルタリングの実装
## 6. 窓関数・STFT

### なぜ必要か

DFTは信号が**定常**（時間的に変化しない）であることを前提とする。 音声・音楽など**時間変化する信号**には対応できない。

また、有限の区間に切り出す操作自体が**矩形窓**の適用であり、スペクトル漏れの原因になる。

### 窓関数（Window Function）

信号に掛け算して端を滑らかに絞る関数。

$$x_w[n] = x[n] \cdot w[n]$$

|窓関数|特徴|用途|
|---|---|---|
|矩形窓|主ローブ幅最小・サイドローブ大|特殊用途のみ|
|Hann窓|バランスが良い|一般的な用途|
|Hamming窓|サイドローブをさらに抑制|音声処理|
|Blackman窓|サイドローブ最小・主ローブ幅大|高精度スペクトル|
|Kaiser窓|パラメータで調整可能|フィルタ設計|

> [!note] トレードオフ
> 
> - 主ローブ幅（周波数分解能）↔ サイドローブレベル（漏れ量）
> - 矩形窓以外は全て周波数分解能を犠牲にして漏れを減らす

### STFT（短時間フーリエ変換 / Short-Time Fourier Transform）

窓をずらしながらDFTを繰り返すことで、**時間×周波数の2次元表現**を得る。

$$\text{STFT}{x[n]}(m, k) = \sum_{n} x[n], w[n - mH], W_N^{kn}$$

- $m$：フレームインデックス（時間方向）
- $H$：ホップサイズ（窓をずらす量）
- $k$：周波数ビン

### スペクトログラム

STFTの振幅の二乗：

$$S(m, k) = |\text{STFT}{x[n]}(m, k)|^2$$

縦軸：周波数、横軸：時間、輝度：パワー → 視覚的に周波数変化を確認できる。

### パラメータのトレードオフ

|パラメータ|大きい場合|小さい場合|
|---|---|---|
|窓長 $N$|周波数分解能↑・時間分解能↓|周波数分解能↓・時間分解能↑|
|ホップサイズ $H$|時間方向が粗い|時間方向が細かい（計算量↑）|

> [!tip] 音声処理の典型値
> 
> - $f_s = 16000$ Hz
> - 窓長：25 ms（400サンプル）
> - ホップサイズ：10 ms（160サンプル）
> - 窓関数：Hann窓

## 7. 変換の関係まとめ

```
z変換 X(z)
    │
    │ z = e^{jω}（単位円に制限）
    ↓
DTFT X(e^{jω})  ←── 理論的な「真のスペクトル」
    │
    │ ω = 2πk/N（N点でサンプリング）
    ↓
DFT X[k]  ←── コンピュータで計算できる近似
    │
    │ 窓をずらしながら繰り返し適用
    ↓
STFT  ←── 時間変化する信号の解析
```

|変換|入力|出力|主な用途|
|---|---|---|---|
|z変換|離散・無限長|複素平面上の関数|安定性・伝達関数|
|DTFT|離散・無限長|連続スペクトル（周期$2\pi$）|理論・周波数特性の定義|
|DFT/FFT|離散・有限$N$点|$N$点の離散スペクトル|実際のスペクトル計算|
|STFT|離散・無限長（時変）|時間×周波数の2次元|音声・非定常信号解析|
## 8. よくある混乱ポイント

> [!faq] DFTとDTFTは何が違う？ DTFTは連続スペクトル（計算機で扱えない）、DFTはそれをN点でサンプリングした近似。DFTは「N点周期信号」という仮定を持つ。

> [!faq] z変換とDTFTはどちらが広い？ z変換の方が広い。DTFTは z=e^{jω}（単位円上）という特殊ケース。単位円上でz変換が収束しないような信号（指数増大など）にはDTFTが存在しないが、z変換は使える。

> [!faq] FFTはDFTと別物？ 同じもの。FFTはDFTを計算するための高速アルゴリズム。O(N²)→O(N log N)。

> [!faq] 窓関数はなぜ必要？ DFTは信号がN点で周期的であると仮定する。実際はそうでないため、端の不連続からスペクトル漏れが生じる。窓関数で端を滑らかにして漏れを軽減する。
