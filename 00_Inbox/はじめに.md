# Inbox

## ここでのルールは1つだけ

**何でもここに、新規ノートで殴り書きする。** 種類・置き場所・テンプレ・タグ・日付・きれいなタイトル——ぜんぶ考えなくていい。授業メモも、論文メモも、実験ログも、思いつきも、まずここ。

書く前に「これは何ノート？どのフォルダ？」と悩まない。**悩む前に手を動かす**のが正解。

## あとで Claude にまかせる

区切りがついたら Claude Code で **`/inbox`** を実行する。Claudeが1枚ずつ「これは○○ノート」と判定して、整形・正しいフォルダへの移動・命名・タグ・リンク/MOC/索引更新まで提案する（実行は承認してから）。

つまり整理の判断はあなたがやらない。あなたは**書くだけ**。

> 振り分け先（参考。実際の判定は `/inbox` が行う）
> - `10_Projects/` 期限・ゴールのある作業　/　`20_Areas/` 継続トピック・知識
> - `30_Resources/` 参考資料・論文　/　`40_Archives/` 完了・不要
