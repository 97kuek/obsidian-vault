# 永続ノート作成

論文メモ、授業ノート、勉強会ノートから、永続ノート（Zettelkasten）の候補を抽出して下書きを作る。

## 判断軸

その概念は「他分野にも通じる横断的な考え方」か。

- Yes: `20_Areas/永続ノート/` に永続ノートを作る。
- No: 科目・トピックノートに残す。

## 原則

- 候補を提案し、承認を得てから作る。
- 1ノート1概念、自分の言葉で書く。
- 下書きには `<!-- 下書き。自分の言葉で書き直すこと -->` を入れる。
- 出典と関連永続ノートにリンクする。
- 統合・削除の前に作業前コミットを提案する。

## 手順

1. `VAULT_INDEX.md`、vault の永続ノート規約、`20_Areas/永続ノート/【MOC】永続ノート.md` を読む。
2. 対象ノートから概念を列挙する。
3. 横断性で仕分ける。
4. 候補を提示し、どれを下書き化するか選んでもらう。
5. `99_Templates/永続ノート用.md` の型で下書きを作る。
6. `【MOC】永続ノート.md` と `VAULT_INDEX.md` を更新する。
7. 切れリンク確認を行い、サマリーを出す。

## 仕分け表

```markdown
| 概念 | 横断性 | 他分野への通じ方 | 行き先 |
|---|---|---|---|
| 探索と活用のジレンマ | ★★★ | 意思決定・経営・研究戦略 | 永続ノート |
| Q学習 | ☆ | 強化学習の手法 | 科目ノート |
```
